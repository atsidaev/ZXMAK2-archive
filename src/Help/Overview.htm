Overview


ZXMak2 is a plugin based ZX Spectrum Virtual Machine. In fact this is a flexible ZX Spectrum emulator.

Each device (including ULA, beeper, tape, etc) is represented as device on the bus.

Project written in C# working on Windows platform and using Managed DirectX.

ZXMak2 supports the following ZX Spectrum clones:
  • ZX Spectrum 48 (contended memory)
  • ZX Spectrum 128 (contended memory)
  • ZX Spectrum +3 (contended memory, but currently without FDD)
  • Pentagon 128/512/1024
  • SCORPION 256
  • ATM 4.50
  • ATM 7.10
  • PROFI 3.xx
  • PROFI 5.xx
  • SPRINTER (except spectrum config)
  • QUORUM 256
  • Leningrad 1
  • LEC 48/528
  • Other (through custom configuration and plugins, for example LEC mod)
